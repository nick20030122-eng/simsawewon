# CSV 요약 대시보드 — 코드 기획서

## 1. 목적

사용자가 CSV 파일을 업로드하면 핵심 숫자 지표를 한눈에 보여 주는 Streamlit 대시보드입니다.

## 2. 핵심 기능

| # | 기능 | 성공 기준 |
|---|------|-----------|
| 1 | CSV 업로드 | `.csv` 파일 1개 업로드 가능 |
| 2 | 숫자 컬럼 요약 | 숫자형 컬럼마다 평균·합계·최대값을 Metrics로 표시 |
| 3 | 막대 그래프 | 첫 번째 숫자 컬럼 기준 상위 10행 막대 차트 표시 |
| 4 | Sidebar | 업로드 파일명, 행·열 개수 표시 |

## 3. UI 요구

- 페이지 제목: **CSV 요약 대시보드**
- 레이아웃: Sidebar + 메인 영역
- 숫자 컬럼이 없으면 안내 메시지 표시 (앱 중단 없음)

## 4. 예외 처리

| 상황 | 기대 동작 |
|------|-----------|
| 파일 미업로드 | "CSV 파일을 업로드해 주세요." 안내 |
| CSV 파싱 실패 | 에러 메시지 표시, 앱 계속 동작 |
| 숫자 컬럼 없음 | "숫자형 컬럼이 없습니다." 안내 |

## 5. 범위 외

- DB 저장, 사용자 로그인, API 연동
- 2개 이상 파일 동시 비교

## 6. 기술

- Streamlit 단일 `app.py`
- pandas 사용 허용
