# 할 일 관리 보드 — 코드 기획서

## 1. 목적

팀원이 할 일을 빠르게 추가·완료 처리할 수 있는 간단한 Streamlit 할 일 관리 앱입니다.

## 2. 핵심 기능

| # | 기능 | 성공 기준 |
|---|------|-----------|
| 1 | 할 일 추가 | 텍스트 입력 + 추가 버튼으로 목록에 등록 |
| 2 | 완료 처리 | 체크박스로 완료/미완료 토글 |
| 3 | Tabs | **미완료** / **완료** 탭으로 분리 표시 |
| 4 | Metrics | Sidebar에 전체·미완료·완료 개수 표시 |
| 5 | 세션 유지 | `st.session_state`로 새로고침해도 목록 유지 |

## 3. UI 요구

- 페이지 제목: **할 일 관리 보드**
- Sidebar: 할 일 통계 Metrics 3개
- 메인: Tabs 2개 (미완료 / 완료)

## 4. 예외 처리

| 상황 | 기대 동작 |
|------|-----------|
| 빈 텍스트로 추가 시도 | 추가하지 않고 안내 메시지 |
| 할 일 0개 | "등록된 할 일이 없습니다." 표시 |

## 5. 범위 외

- 사용자 로그인, DB, 알림, 우선순위/마감일 기능
- 외부 API 연동

## 6. 기술

- Streamlit 단일 `app.py`
- 추가 라이브러리 없음 (표준 라이브러리 + Streamlit만)
